# Gupshup Restaurant Website

## Overview
A modern, responsive restaurant branding website for **Gupshup** - "Good Food, Better Company". This website showcases the restaurant's menu, ambience, offers, and provides links to order from Zomato and Swiggy.

## 🎯 Purpose
This is a **branding website only** - designed to:
- Showcase the restaurant's identity and atmosphere
- Display menu highlights and special dishes
- Promote current offers and deals
- Direct customers to Zomato & Swiggy for online ordering
- Provide contact information and location

**Note:** This website does NOT include direct ordering, cart, or payment functionality.

## 📁 Files
- `index.html` - Main HTML structure
- `style.css` - Complete CSS styling
- `script.js` - JavaScript for interactivity

## 🚀 How to Use

### Local Development
1. Download all three files to a folder
2. Open `index.html` in a web browser
3. No server or build process required!

### GitHub Deployment
1. Create a new repository on GitHub
2. Upload all three files: `index.html`, `style.css`, `script.js`
3. Go to Settings → Pages
4. Select main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Customization
- **Logo:** Replace the logo URL in `index.html` with your own
- **Colors:** Modify CSS variables in `:root` section of `style.css`
- **Menu Items:** Update menu section in `index.html`
- **Images:** Replace Unsplash URLs with your own images
- **Contact Info:** Update phone, address, and social media links
